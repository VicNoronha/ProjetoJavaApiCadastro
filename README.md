



### API de Cadastro de Usuários

Projeto  desenvolvido em Java com Spring Boot 



###  Sobre o projeto

Esta API tem como objetivo realizar o cadastro e gerenciamento de usuários, aplicando conceitos fundamentais de desenvolvimento backend, Programação Orientada a Objetos e arquitetura REST.




###  Objetivo

O principal objetivo deste projeto é:

 Praticar a criação de uma API REST em Java
 Aplicar conceitos de CRUD (Create, Read, Update, Delete)
 Trabalhar boas práticas de organização de código
 Consolidar o uso do Spring Boot para desenvolvimento backend
 Preparar uma base sólida para projetos mais complexos no futuro


###  Funcionalidades

Cadastrar usuário;

Atualizar dados do usuário;

Excluir usuário;

Buscar usuário por ID;

Listar todos os usuários cadastrados;
